myFlix API

myFlix API provides users with access to information about different movies, directors, and genres. The API also allows users to register and save data about their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies GET none A JSON object containing data about all movies
Get data about a single movie by title /movies/[Title] GET none A JSON object holding data about a single movie, containing a description, genre, director, image URL, and whether it is featured or not. Example:
{
id: 1,
Title: "Silence of the Lambs",
Description: "A young FBI cadet must receive the help of an incarcerated and manipulative cannibal killer to help catch another serial killer.",
Genre: {
Name: "Thriller",
Description: "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience."
},
Director: {
Name: "Jonathan Demme",
Bio: "Robert Jonathan Demme was an American director, producer, and screenwriter.",
Birth: "1944",
Death: "2017"
},
ImagePath: "silenceofthelambs.png",
Featured: true
}
Get data about a genre by name/title /movies/genres/[Genre] GET none A JSON object holding data about a particular genre. Example:
Genre: {
Name: "Thriller",
Description: "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience."
}
Get data about a director by name /movies/directors/[Name] GET none A JSON object holding data about a single director containing a bio, birth year, and death year. Example:
Director: {
Name: "Jonathan Demme",
Bio: "Robert Jonathan Demme was an American director, producer, and screenwriter.",
Birth: "1944",
Death: "2017"
},
Allow new users to register /users POST a JSON object holding data about the user being registered. Example:
{
Username: 'movielover123'
Password: 'password1'
Email: movielover@gmail.com
Birthday: '1995-12-31'
} 		A JSON object holding data about the user that was registered including a unique ID. Example:
{
ID: 123abc456,
Username: 'movielover123'
Password: 'password1'
Email: movielover@gmail.com
Birthday: '1995-12-31'
}
Update user info by username /users/[Username] PUT A JSON object holding the user data that should be updated A JSON object containing the updated user info
Add a movie to list of favorites /users/[Username]/movies/[MovieID] POST none A JSON object holding data about the user including the updated list of favorite movies. Example:
ID: 123abc456,
Username: 'movielover123'
Password: 'password1'
Email: movielover@gmail.com
Birthday: '1995-12-31'
FavoriteMovies: ["60a73ce9eba4f336ba54a2f3"]
}
Remove a movie from list of favorites /users/[Username]/movies/[MovieID] DELETE none A JSON object holding data about the user including the updated list of favorite movies.
Allow users to deregister /users/[Username] DELETE none A text message indicating that the user email was removed successfully
Allow users to login /login POST none A JSON object holding data about the user logging in along with a JWT token for authentication
